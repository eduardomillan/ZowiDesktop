Zowi Desktop — Guia d'usuari

Zowi Desktop és una aplicació multiplataforma per controlar i programar el teu robot Zowi des de l'ordinador mitjançant Bluetooth. Aquesta guia et mostra cada pantalla de l'aplicació.

1. Trieu el vostre idioma

En obrir l'aplicació veus la pantalla d'inici. Trieu el vostre idioma a la llista — Español, Valencià, English, Français, Български — i premeu Continuar. També podeu Sortir de l'aplicació des d'aquí.

Pantalla d'inici amb selector d'idioma

2. Benvinguda

La pantalla de benvinguda presenta el Zowi. Premeu Començar per iniciar la configuració, o Més informació per llegir sobre el robot.

Pantalla de benvinguda

3. Engega el teu Zowi

L'assistent de configuració comprova que el teu Zowi està a punt. Assegura't que està ences i a prop. Si ja està ences, tria “Ja està ences”. Si encara no tens un Zowi, també pots explorar l'aplicació sense ell.

Assistent de configuració — engega el Zowi

4. Connecta't per Bluetooth

L'aplicació busca dispositius Zowi propers. Quan el teu Zowi aparega a la llista, prem-lo. Se't demanarà que introduïsques el codi d'aparellament 1234 per aparellar-lo amb el robot.

Buscant Zowis
Aparellar amb el teu Zowi (codi 1234)

5. Anomena el teu Zowi

Escriu un nom per al teu Zowi i prem Anomena. El nom s'envia al robot i es recorda la propera vegada.

Anomenant el teu Zowi

6. Pantalla principal

Un cop connectat arribes a la pantalla principal, que té dues pàgines:

  • Aplicacions Zowi — jocs i modes interactius.
  • Projectes — activitats de programació.

Fes servir els botons superiors per obrir Ajustos i Assoliments. La barra d'estat superior mostra l'estat de connexió:

  • Connectat · <nom> · <bateria>% · <MAC> — el teu Zowi està a punt.
  • Connectant… — l'aplicació està intentant localitzar el teu Zowi.
  • Zowi no disponible — el teu Zowi guardat està apagat o fora d'abast; prem Reconecta.
  • No connectat — no s'ha configurat cap Zowi.
Inici — Aplicacions Zowi
Inici — Projectes

7. Reconnexió automàtica

Cada vegada que obres l'aplicació, Zowi Desktop intenta reconnectar amb l'últim Zowi usat si està ences. Si està apagat, la barra d'estat t'ho indicarà i podràs prémer Reconecta en qualsevol moment.

8. Restablir (mode desenvolupador)

El botó Restablir, visible en mode desenvolupador, elimina el Zowi registrat de l'aplicació i del sistema Bluetooth. Si no hi ha cap Zowi registrat, es mostra un missatge d'avís.
